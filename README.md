## About The Project

달팽이키우기, 다마고치에서 많은 영감을 받았습니다. 체력 감소, 죽음, 진화, 밥주기, 놀아주기, 행동력 리필 등 기본 기능이 구현되어 있으며 캐릭터는 Spine을 사용해 구현했습니다.
